# Day 11 – ATS Resume Optimizer & Resume Generator

## Objective
Optimize my resume for ATS (Applicant Tracking System) compatibility using Claude AI based on a target Job Description.

## Tools Used
- Claude AI
- ATS Resume Optimizer Prompt
- Sample Job Description
- GitHub

## What I Did
- Uploaded my existing resume.
- Used a sample Data Analyst Job Description.
- Ran the ATS Resume Optimizer prompt in Claude AI.
- Reviewed the ATS Match Score and Gap Analysis.
- Generated an ATS-friendly optimized resume.
- Downloaded the optimized resume in DOCX format.

## ATS Analysis
- ATS Match Score: ~45/100
- Missing Keywords:
  - Data Analysis
  - Data Visualization
  - SQL Reporting
  - Dashboards
  - KPIs
  - Data Cleaning & Validation
- Missing Skills:
  - Excel
  - Tableau
  - Statistics
  - ETL Concepts

## Key Learnings
- ATS resumes should be tailored to the target Job Description.
- Proper keyword alignment improves resume visibility.
- Skills and project descriptions should match the job requirements.
- Honest optimization is better than adding fake experience.
