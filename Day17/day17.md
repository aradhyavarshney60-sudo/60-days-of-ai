# Day 17 – AI Fuel Economics Dashboard

## Objective

Build an interactive Fuel Economics Dashboard using Claude AI and a CSV dataset to compare different fuel types based on running cost, maintenance cost, emissions, and overall efficiency.

---

## Project Overview

In this project, I used Claude AI to analyze a fuel economics dataset and generate a fully interactive HTML dashboard. The dashboard helps compare different fuel types based on cost, environmental impact, and vehicle ownership factors.

---

## Vehicle Details

- **Vehicle Model:** Maruti Suzuki Swift
- **Fuel Type:** Petrol
- **Usage:** Mixed
- **Monthly Distance:** 1000 km
- **Vehicle Age:** 3 Years

---

## Dashboard Features

- Interactive Fuel Economics Dashboard
- Cost per Kilometer Analysis
- Monthly Fuel Cost Estimation
- CO₂ Emission Comparison
- Fuel Cost vs Vehicle Age
- E85 Compatibility Score
- Fuel Type Comparison Cards
- Data-driven Recommendations

---

## Key Insights

- Petrol offers balanced performance for mixed usage.
- E85 provides lower emissions but requires a compatible engine.
- CNG delivers lower running costs for high-mileage users.
- Diesel is suitable for long-distance highway driving.
- Electric Vehicles have the lowest running cost but depend on charging infrastructure.
