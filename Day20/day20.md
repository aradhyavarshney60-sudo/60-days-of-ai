# Day 20 – AI Face Puzzle Game

## Objective

Build an AI-powered Face Puzzle Game using Claude AI. The application captures a photo using the webcam, converts it into a puzzle, supports multiple difficulty levels, and tracks completion time and moves.

---

## Features

- Webcam Integration
- Capture Photo
- 3×3, 4×4 & 5×5 Difficulty
- Drag & Drop Puzzle
- Timer
- Move Counter
- Win Detection
- Leaderboard
- Local Storage
- Responsive UI

---

## Technologies Used

- HTML
- CSS
- JavaScript
- Canvas API
- MediaDevices API
- Local Storage

---

## Learning Outcomes

- Browser Camera API
- Canvas Image Processing
- Drag & Drop API
- Local Storage
- Interactive Game Development
