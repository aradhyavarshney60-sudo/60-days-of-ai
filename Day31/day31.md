# Day 31 – AI Supply Chain Control Tower

## Overview
Today I built an AI-powered Supply Chain Control Tower simulation using HTML, CSS, and Vanilla JavaScript. The application simulates real-world operational disruptions where users act as the Head of Operations and make decisions that directly affect key business KPIs.

## Features
- Interactive Supply Chain Control Tower
- Live KPI Dashboard
- Real-time Operational Alerts
- Countdown Timer
- Decision-based Gameplay
- Dynamic Score System
- Performance Summary Dashboard
- Responsive Dark UI
