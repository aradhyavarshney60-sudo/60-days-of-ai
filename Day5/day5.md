# Day 5 – Context is King

## Objective

Today I learned how adding context completely changes the quality of AI responses.

## Prompt A (Without Context)

I asked Claude to create a 30-day learning roadmap without providing any personal information.

### Observation

The roadmap was useful but generic. It was designed for any beginner and lacked personalization.

---

## Prompt B (With Context)

I provided my background:

- B.Tech Student at AIMT
- Skills: Python, C++, SQL, Git & GitHub
- Goal: Become an AI/ML Engineer
- Available Time: 4 Hours per Day
- Learning Style: Hands-on Projects

### Observation

The roadmap became much more personalized and included relevant projects, GitHub goals, learning milestones, and practical resources.

---

## Comparison

### Which roadmap feels more personalized?

Prompt B because it considers my current skills, learning style, available time, and career goal.

### Which roadmap would I actually follow?

Prompt B because it is practical, realistic, and aligned with my AI career journey.

### What role did context play?

Providing context allowed Claude to generate a roadmap specifically tailored to my needs instead of a generic plan.

---

## Key Learnings

- Context improves AI responses.
- Personalized prompts generate better results.
- The more information you provide, the better AI understands your requirements.
- Prompt Engineering is not just asking questions; it is giving AI the right context.
