# Day 7 - Claude Model Selection & Reasoning Effort

## Objective
Learned how to choose the right Claude model and reasoning effort for different tasks and created my personalized Claude usage strategy.

## What I Learned
- Difference between Haiku, Sonnet, and Opus models
- When to use Low, Standard, High, and Max reasoning effort
- How to select the right model based on the task
- Creating an efficient AI workflow for learning and coding

## My Personalized Claude Strategy

### Recommended Primary Model
**Sonnet** for everyday learning, coding, and writing tasks.

### When to Use Haiku
- Quick summaries
- Simple questions
- Fast responses

### When to Use Sonnet
- Coding
- Learning AI/ML
- Writing documentation
- GitHub README files

### When to Use Opus
- Deep research
- Complex reasoning
- Long-term planning

### Recommended Reasoning Effort
- **Low:** Quick tasks
- **Standard:** Daily work
- **High:** Difficult coding problems
- **Max:** Advanced research and complex projects

## Key Takeaways
Choosing the correct Claude model and reasoning effort improves productivity, response quality, and efficiency.
